# SavorGlobal - Culinary Guide

A premium React application for discovering global cuisines and recipes.

## Features
- Search for recipes by name or ingredient.
- Filter by cuisine type.
- View detailed recipe instructions and ingredients in a beautiful modal.
- Responsive, modern design with glassmorphism and smooth animations.

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open in browser**:
   Navigate to `http://localhost:5173`.

## Technologies
- **Vite + React** for fast development and rendering.
- **Vanilla CSS** with CSS Variables for flexible, high-performance styling.
- **Lucide React** for beautiful icons.
